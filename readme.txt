== Description ==
<a href="happyplugins.com">HappyPlugins</a> present <a href="http://wlmp.info/wlm">Wishlist Member</a> Dojo plugin.

<p>Wishlist Dojo is like a directory that contains all the Wishlist Member dedicated plugins in one place!<br><br>
Wishlist Dojo Currently Contains 48 plugins & 9 Premium Guides!</p>

Features:<br><ul>
<li><strong>Display only the plugins you need</strong>: Wishlist Dojo can be filtered to display only the types of plugins you need, for example: Widgets, Management plugins, Shopping carts related plugins etc.</li>
<li><strong>Know which plugins you already own and save time</strong>: Wishlist Dojo is displays an “installed” message for
all the plugins in the list you already own and installed in your site.</li>
<li><strong>Direct link to purchase your desired plugins:</strong> Wishlist Dojo redirects you to the plugin or guide’s sales page
straight from the Wishlist Dojo plugin itself.</li>
</ul>
<br>
<strong>How Can Wishlist Dojo Help Empower Your Membership Site?<strong></br>
<p>Wishlist Dojo shows you all the available plugins for Wishlist Member, so you can easily find the plugins you need.<br><br>
For example: a plugin that creates a visual members’ dashboard, a quiz plugin, an auto registration plugin & a lot more!</p>

<strong>Will Wishlist Dojo Directory Update With New Plugins?</strong><br>
<p>Of course! The directory updates with all the new plugins from time to time (usually once a month).<br><br>
You will get an update by email every time a new version is released.</p>

<strong><a href="http://wlmp.info/wlm">Wishlist Member</a> is required to use Wishlist Dojo</strong><br>

== Installation ==
1. Upload the plugin to "wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
